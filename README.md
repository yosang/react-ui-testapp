# VORTEX Sportswear E-Commerce Frontend
A modern, responsive e-commerce frontendr. Built as a portfolio project to demonstrate the capabilities of my custom React UI component library.

DEMO: https://vortex-e-commerce-7ldu.vercel.app/

# About

This is a e-commerce frontend for a fictional sportswear store, designed with clean layouts, responsive grids, and an intuitive user experience. It showcases a shopping interface built entirely with my custom React component library `@yosang/react-ui`.

# Features
- Responsive design for mobile, tablet, and desktop
- Hero section that supports image and video
- Sticky Navbar with multi-level navigation and icons
- Drawer system for sidebar menu and shopping cart
- Login modal with form inputs
- ProductCard component with discount, title, description, and price
- Brand showcase grid

# Technologies
- React
- React Router
- [@yosang/react-ui](https://www.npmjs.com/package/@yosang/react-ui)
- Vite
- Custom UI Components (Hero, ProductCard, Drawer, Modal, Navbar, Container, Input, Button, etc.)

### Installation & Usage

1. Clone the project with `https://github.com/yosang/vortex-e-commerce.git`
2. Navigate to the project directory with `vortex-e-commerce`
3. Installa required packaged with `npm install` 
4. Run the development environment with `npm run dev`
